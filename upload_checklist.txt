========================================
UPLOAD INSTRUCTIONS
========================================

1. 📂 FOLDER STRUCTURE
   - Upload all files maintaining the folder structure
   - Don't change folder names
   - Keep file permissions as 644 for files
   - Keep folder permissions as 755 for directories

2. 🔧 CONFIGURATION UPDATES
   - Update config.php with your hosting details
   - Update database credentials
   - Update site URL
   - Update email settings

3. 📊 DATABASE SETUP
   - Import ihome.sql to your database
   - Or run database_schema.sql for fresh setup
   - Test database connection

4. 🔒 SECURITY SETTINGS
   - Set proper file permissions
   - Update admin password
   - Enable SSL certificate
   - Configure .htaccess

5. 📧 EMAIL TESTING
   - Test contact form
   - Test calculator form
   - Verify email delivery

6. 🧪 FUNCTIONALITY TESTING
   - Test all pages load correctly
   - Test forms submission
   - Test image galleries
   - Test mobile responsiveness

========================================
TROUBLESHOOTING
========================================

❌ Database Connection Error
   - Check database credentials in config.php
   - Verify database exists
   - Check database host

❌ 500 Internal Server Error
   - Check .htaccess syntax
   - Verify PHP version compatibility
   - Check file permissions

❌ Images Not Loading
   - Check file paths
   - Verify image files uploaded
   - Check file permissions

❌ Forms Not Working
   - Check email configuration
   - Verify form action URLs
   - Test database connection
